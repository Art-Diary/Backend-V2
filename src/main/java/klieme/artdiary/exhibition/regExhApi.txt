-----이혜원--------
(기능) 등록할 전시회 추가(사용자)
- method: post
- url: /regexh
- request data: 제목, 장소, 일정, 작가, 관람료, 홈페이지 링크(null), 소개(null), 포스터
{
    String regExhName;
    String regGallery;
    LocalDate regExhPeriodStart;
    LocalDate regExhPeriodEnd;
    String regPainter;
    String regFee;
    String regIntro; (nullable)
    String regUrl; (nullable)
    MultipartFile regPoster;
    String regArt; (nullable)
    LocalDateTime regDate;
}
- response data: no., 전시회 제목, 등록자, 등록일, 등록 현황
{
        Long regExhId;
        String regExhName;
        String regGallery;
        LocalDate regExhPeriodStart;
        LocalDate regExhPeriodEnd;
        String regPainter;
        String regFee;
        String regIntro;
        String regUrl;
        String regPoster;
        String regArt;
        String regComment;
}
------------------

(기능) 등록할 전시회 목록 조회(사용자/관리자)
- method: get
- url: /regexh?isAdmin=true/false
- request data: no
- response data: no., 전시회 제목, 등록자, 등록일, 등록 현황
list: [
    Long no;
    Long regExhId;
    String regExhName;
    String regNickName;
    LocalDateTime regDate;
    Boolean regState;
]
